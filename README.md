## 🎙️ VoiceMail

Mensagens diretas com quem você conhece.

App Android que carrega o VoiceMail em WebView.

### 📦 Build

O APK é gerado automaticamente pelo GitHub Actions ao fazer push na branch `main`.

### 🚀 Instalação

1. Vá em **Actions** > clique no workflow mais recente
2. Baixe o **VoiceMail-APK**
3. Instale no seu celular
